# ztree-parse-md

GFM Markdown parser for ztree. Parses Markdown text into a ztree `Node` tree.

## Quickstart

```bash
git clone git@github.com:erwagasore/ztree-parse-md.git
cd ztree-parse-md
zig build        # build library
zig build test   # run tests
```
